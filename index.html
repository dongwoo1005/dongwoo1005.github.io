Personal Project


Square Swapper 5000

Basically, it's Candy Crush Saga.

A match-three style puzzle game. The game is coded in C++, designed and implemented in OOP way. Check out its UML and design document for details.
The controls are done through UNIX command line. The goal of the game is to beat the levels by meeting the requirements such as obtaining certain amount of scores, unlocking all the locked squares within a number of provided moves. The score goes up as you make matches of 3 or more squares. Available commands are: swap, hint, scramble, levelup, leveldown, restart. Using the command line arguments, you may run the game in text mode only, change the seed number, provide an initial board configuration, set start level, and turn on testing mode.


imageCompression

It takes input of an original image and the drop tolerance number to compute a compressed image. Computation was done by dropping the Fourier coefficient having modulus less than provided tolerance number to 0 from every 8 x 8 subblock for each colour.


Flood-It!

A copy of the game Flood-It!, written in C++. The controls are done through UNIX command line. You may customize the grid board size, the number of initially provided moves in my version and the initial board configuration. The object of the game is to flood-fill the entire board with the same colour.
