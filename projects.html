Square Swapper 5000

A match-three style puzzle game that draws elements from popular games, such as Candy Crush Saga and the Bejewelled Series. The game is coded in C++. The controls are done through UNIX command line. The goal of the game is to beat the levels by meeting the requirements such as obtaining certain amount of scores, unlocking all the locked squares within a number of provided moves. The score goes up as you make matches of 3 or more squares. Available commands are: swap, hint, scramble, levelup, leveldown, restart. Using the command line arguments, you may run the game in text mode only, change the seed number, provide an initial board configuration, set start level, and turn on testing mode.


Flood-It!

A copy of the game Flood-It! (http://floodit.appspot.com/), written in C++. The controls are done through UNIX command line. You may customize the grid board size, the number of initially provided moves in my version and the initial board configuration. The object of the game is to flood-fill the entire board with the same colour.
